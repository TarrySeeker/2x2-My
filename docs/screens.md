Страница **одной услуги** с тремя возможными сценариями покупки

### 2.4 Варианты по `pricing_mode`

#### A. `fixed` (визитки с фиксированной ценой)

Правая колонка:
```
от 1 700 ₽
[В корзину]   [Купить в 1 клик]
```
Без калькулятора. Quantity — опционально (один `QuantityStepper`).

#### B. `calculator` (баннеры, буквы, листовки)

Правая колонка:
```
от 5 100 ₽  ← пересчитывается live
┌─ Рассчитать стоимость ────┐
│ Ширина:  [3.0] м           │
│ Высота:  [2.0] м           │
│ Площадь: 6.00 м²           │
│ Цена/м²: 850 ₽             │
│ ──────────                 │
│ Итого:   5 100 ₽           │
└────────────────────────────┘
[В корзину]   [Купить в 1 клик]
```

Под калькулятором — ссылка «Нестандартные размеры? Заказать расчёт»
→ открывает `QuoteModal` с предзаполнением параметров.

#### C. `quote` (стелы, фасады — только по запросу)

Правая колонка:
```
По запросу
[ Заказать расчёт ]   ← primary, w-full
[ Позвонить +7-932-424-77-40 ]   ← secondary
```

Без калькулятора и без корзины. Только модалка `QuoteModal`.

### 2.5 `ProductGallery` — состояния

| Состояние | Что |
|---|---|
| Одно фото | Только main image, без thumbnails |
| 2–6 фото | Main + thumbnails ряд |
| 7+ фото | Main + thumbnails + «+N» кнопка → lightbox |
| 0 фото | Placeholder SVG с оранжевой пиктограммой камеры |

### 2.6 Состояния CTA

| Условие | Главный CTA |
|---|---|
| `pricing_mode=fixed` + `in_stock=true` | «В корзину» (primary) |
| `pricing_mode=fixed` + `stock<1` | «Нет в наличии» (disabled) + «Уведомить» |
| `pricing_mode=calculator` + все поля заполнены | «В корзину» (primary, с итоговой ценой) |
| `pricing_mode=calculator` + поля пустые | «В корзину» (disabled) + tooltip «Заполните параметры» |
| `pricing_mode=quote` | «Заказать расчёт» (primary) |

## 4. Overlay компоненты

### 4.1 `OneClickModal`

См. `docs/design-system.md` п. 19. Anatomy:
- `max-w-md`, `rounded-2xl`, `p-8 pt-10`
- Close button `top-4 right-4`
- Заголовок Rubik 700 `text-2xl`
- Форма: name, phone, (скрытый hidden) product_id, product_name, price
- Disclaimer внизу (политика)
- Primary CTA `w-full rounded-full`

**Trigger:** кнопка «Купить в 1 клик» в `ProductInfo`.
**Submit:** `POST /api/leads` → toast + close.

### 4.2 `QuoteModal`

См. `docs/design-system.md` п. 20. Anatomy:
- `max-w-lg`, прокручивается внутри (`max-h-[90dvh]`)
- Поля:
  - `name` (required)
  - `phone` (required)
  - `email` (optional)
  - Параметры услуги: `width`, `height`, `material`, `address` (если есть
    контекст из калькулятора — префилл)
  - `comment` (textarea)
  - `file` (опционально, drag-drop)
- Primary CTA «Отправить заявку»

**Trigger:** разные:
- Клик «Заказать расчёт» на карточке `pricing_mode=quote`
- Клик «Нестандартные размеры» в калькуляторе
- Клик в CTA-полоске каталога «Не нашли нужное?»
- Клик в footer/header «Расчёт»

**Submit:** `POST /api/quote` → запись в `calculation_requests` →
toast + close + analytics event `quote_request`.

### 4.3 Lightbox (галерея карточки)

Radix `Dialog` fullscreen, `bg-black/90`, Embla Carousel внутри.
- `Esc` для закрытия
- Клик вне слайда — закрытие
- Стрелки `←` `→` для навигации
- Индикатор `3 / 8` сверху
- Кнопка «Скачать» — **нет** (Этап 4, если попросят)

Только для `≥lg` — на мобиле Lightbox избыточен (галерея уже фуллскрин).

## 7. A11y чеклист для каждого экрана

- [ ] **Каталог:** `<h1>` один, `<nav aria-label="...">` на всех nav, focus-visible на карточках, keyboard-навигация по чипсам стрелками
- [ ] **Фильтры:** каждый checkbox связан с `<label>`, группы — `<fieldset><legend>`, слайдер цены — aria-valuetext с форматированной ценой
- [ ] **Карточка товара:** `<h1>` название, `<figure>` для галереи + `<figcaption>`, калькулятор — полноценная форма с aria-describedby для итогов
- [ ] **Модалки:** Radix Dialog — автоматический focus-trap, Esc close, aria-labelledby на title, aria-describedby на subtitle
- [ ] **Toast:** `role="status"`, `aria-live="polite"`
- [ ] **Sticky mobile bar:** `role="complementary"`, кнопка с aria-label «Добавить в корзину»
- [ ] **Контраст:** все тексты ≥ 4.5:1, ссылки underline или focus-ring на ≥ 3:1
- [ ] **Prefers-reduced-motion:** все framer-motion завёрнуты в `useReducedMotion()`
